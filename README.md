# rpchelper

rpchelper определяет активное окно и показывает информацию о нем в discord rich presence.

сторонние python-пакеты не нужны: используется только стандартная библиотека python.

## возможности

- определяет активное окно через x11 на linux и winapi на windows
- читает класс окна, заголовок и имя процесса
- отправляет активность напрямую в discord ipc (unix socket на linux, named pipe на windows, без `pypresence`)
- обновляет rich presence только при смене активного окна

## требования

- linux с x11-сессией или windows
- python 3.9+
- запущенное десктопное приложение discord
- для linux: наличие x11 runtime-библиотеки (`libX11.so.6`, обычно уже установлена)

## запуск

```bash
./run.sh
```
или

```bat
run.bat
```

оба скрипта запуска можно вызывать прямо из корня проекта. если есть `.venv`, будет использован он.

## примечания

- эта сборка рассчитана на x11. в wayland определение активного окна зависит от композитора и может не работать.
- если discord не запущен, приложение завершится с ошибкой подключения к ipc.
