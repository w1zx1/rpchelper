# rpchelper

rpchelper определяет активное окно в linux x11 и показывает информацию о нем в discord rich presence.

сторонние python-пакеты не нужны: используется только стандартная библиотека python.

## возможности

- определяет активное окно через x11 (`ctypes` + `libX11`)
- читает класс окна, заголовок и имя процесса
- отправляет активность напрямую в discord ipc (без `pypresence`)
- обновляет rich presence только при смене активного окна

## требования

- linux с x11-сессией
- python 3.9+
- запущенное десктопное приложение discord
- наличие x11 runtime-библиотеки (`libX11.so.6`, обычно уже установлена)

## настройка

1. создай приложение в discord: https://discord.com/developers/applications
2. скопируй **Application ID** (это и есть твой client ID).

## запуск

```bash
./run.sh
```

## примечания

- эта сборка рассчитана на x11. в wayland определение активного окна зависит от композитора и может не работать.
- если discord не запущен, приложение завершится с ошибкой подключения к ipc.
